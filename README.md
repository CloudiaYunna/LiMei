*受够了像<br>

本插件专为[上海立信会计金融学院教务系统](https://lxjw.lixin.edu.cn)开发，旨在解决官方教务系统界面老旧、视觉体验不佳、操作不够便捷的问题。

意在**不影响系统原有所有功能**（如选课、查分、课表查询）的前提下，实现界面美化、操作简化，提升立信学子使用教务系统的舒适度与效率。

# ❌ 常见问题

- **我可以在手机、平板上用吗？**
  
  通常**不可以**，本浏览器插件只支持Chromium内核的浏览器（如Google Chrome、Microsoft Edge）。你可能会看到在手机上通过油猴脚本管理器或脚本猫安装浏览器插件的教程，但本插件目前并没有这种安装方法。~~以后等作者学了说不定会搞（~~

- **这玩意会收集我的账号密码吗？**

  **不会。** 本插件仅修改浏览器前端渲染样式与少量交互，不篡改教务系统任何后端数据、也完全不收集、不存储任何用户信息（登录账号、密码、成绩等），所有操作均在本地浏览器完成。~~代码摆在那里自己去看我有没有这个能耐~~
