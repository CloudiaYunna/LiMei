本插件专为[上海立信会计金融学院教务系统](https://lxjw.lixin.edu.cn)开发，旨在解决官方教务系统界面老旧、视觉体验不佳、操作不够便捷的问题。

意在**不影响系统原有所有功能**（如选课、查分、课表查询）的前提下，实现界面美化、操作简化，提升立信学子使用教务系统的舒适度与效率。
